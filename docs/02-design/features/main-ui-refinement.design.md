# 설계서: 메인 화면 UI 고도화 (Flow Free 스타일)

## 1. UI 컴포넌트 상세 설계

### 1.1 메인 로고 (Logo)
- **텍스트**: "flow"
- **스타일**:
    - 'f': `#FF0000` (Red)
    - 'l': `#00FF00` (Green)
    - 'o': `#0000FF` (Blue)
    - 'w': `#FFFF00` (Yellow)
- **폰트**: 120px, Bold, Sans-serif.
- **위치**: (300, 250) 중앙 상단.

### 1.2 메인 메뉴 리스트 (`UIManager.createMenuListItem`)
- **공통 스타일**:
    - 항목 높이: 80px.
    - 좌측 텍스트: 흰색, 42px, (x: 100).
    - 우측 요소: (x: 500).
- **항목 구성**:
    1. **무료 플레이**: 우측에 흰색 '>' (42px).
    2. **일일 퍼즐**: 우측에 빨간색 배지 + 숫자 '6'.
    3. **주간 퍼즐**: 우측에 파란색 배지 + 숫자 '30'.
    4. **타임 트라이얼**: 우측에 스톱워치 아이콘 (또는 '⏱' 기호).

### 1.3 배지 디자인 (`UIManager.createBadge`)
- **형태**: 리본 모양 또는 끝이 뾰족한 직사각형.
- **크기**: 60x40.
- **텍스트**: 중앙 정렬, 흰색, 24px.

### 1.4 하단 유틸리티 버튼 (`UIManager.createUtilityButton`)
- **구성**: 아이콘(상) + 텍스트(하).
- **위치**:
    - 광고 제거: (120, 720).
    - 스토어: (300, 720).
    - 다른 게임: (480, 720).
- **아이콘 스타일**: 32px 크기, 흰색/유색 조합.

## 2. Layout & Workflow

### 2.1 `TitleScene.js` 리팩토링
1. **Logo 배치**: `createMultiColorTitle`을 활용하여 "flow" 렌더링.
2. **Settings 버튼**: 우측 상단 (550, 50)에 톱니바퀴 아이콘 배치.
3. **Menu 렌더링**:
    - `y` 좌표 420부터 시작하여 90px 간격으로 항목 배치.
4. **Bottom Bar**: 화면 하단에 유틸리티 버튼 3종 배치.

### 2.2 시각적 효과
- **배경**: `0x000000` (Solid Black).
- **그라데이션**: 로고 뒷부분에 아주 연한 멀티 컬러 글로우 효과 (`Graphics.fillGradientStyle` 또는 알파값이 낮은 원형 그래픽 조합).
