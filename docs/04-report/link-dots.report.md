- **프로젝트 명**: LinkDots
- **목표**: Phaser 3를 이용한 Flow Free 스타일의 퍼즐 게임 핵심 메커니즘 구현

## 2. 주요 성과 및 구현 기능
### 2.1 핵심 메커니즘
- **그리드 시스템**: 5x5 가변 그리드와 셀 기반 상태 관리 시스템 구축.
- **경로 그리기**: 부드러운 드래그 인터랙션과 실시간 라인 렌더링 구현.
- **교차 로직**: 선이 겹칠 때 기존 경로를 지능적으로 끊어주는 로직 구현.
- **백트래킹**: 이미 그린 경로 위로 되돌아갈 때 경로가 자동으로 짧아지는 기능 구현.

### 2.2 UI/UX 및 게임성
- **실시간 HUD**: 이동 횟수(Moves)와 보드 점유율(Flow %) 실시간 업데이트.
- **승리 판정**: 모든 점 연결 및 그리드 100% 점유 시 'PERFECT' 메시지 표시.
- **초기화 기능**: 레벨 리셋 버튼을 통해 상태 초기화 지원.
