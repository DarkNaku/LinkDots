# 계획서: 패키지 선택 화면 UI 고도화 (Flow Free 스타일)

## 1. 개요
제공된 이미지(`package_scene.png`)를 분석하여, 현재의 기본적인 패키지 선택 화면을 카테고리별 리스트 형태의 세련된 UI로 고도화합니다.

## 2. 핵심 디자인 요소 (이미지 분석 반영)
- [ ] **헤더 (Header)**:
    - 좌측 상단: 원형 테두리의 뒤로가기 버튼 (`createCircularBackButton` 재사용).
    - 중앙 상단: "레벨" 타이틀 (글자별 색상 다름: '레'는 빨간색, '벨'은 녹색 계열).
- [ ] **프로모션 문구**: 타이틀 아래 작은 텍스트로 안내 문구 표시.
- [ ] **카테고리 섹션 (Category Sections)**:
    - **섹션 헤더**: 가로로 꽉 찬 유색 배경(녹색, 빨간색, 파란색 등)에 흰색 텍스트로 카테고리명 표시.
    - **패키지 리스트**: 각 섹션 하단에 해당 카테고리의 패키지들을 리스트 형태로 배치.
- [ ] **패키지 항목 (Pack Items)**:
    - 좌측: 패키지 이름 (해당 카테고리 테마 색상 적용).
    - 우측: 진행률 표시 (예: "0 / 150").
    - 상호작용: 항목 클릭 시 해당 패키지의 레벨 선택 화면으로 이동.
- [ ] **배경 (Background)**:
    - 완전한 검은색 배경에 섹션 색상과 매칭되는 세로형 은은한 그라데이션(Glow) 효과.

## 3. 기술 스택 및 구현 전략
- **UI 구성**: Phaser 3 `Graphics` 및 `Text`를 조합하여 리스트형 레이아웃 구현.
- **스크롤 시스템**: 패키지가 많아질 경우를 대비하여 컨테이너 기반의 스크롤 로직 고려.
- **데이터 바인딩**: `levels.json` 구조를 카테고리(packs 내 그룹화)를 지원하도록 확장하거나 현재 구조에서 동적으로 분류.

## 4. 상세 작업 목록
- [ ] **UIManager 확장**:
    - `createSectionHeader(y, text, color)`: 섹션 구분 바 생성 메서드.
    - `createPackListItem(y, name, progress, color, callback)`: 리스트 항목 생성 메서드.
- [ ] **PackSelectScene 리팩토링**:
    - 기존 버튼 나열 방식에서 카테고리별 리스트 방식으로 전환.
    - `levels.json`의 데이터를 섹션별로 렌더링하는 루프 구현.
- [ ] **시각적 효과**: 타이틀 색상 개별 적용 및 배경 글로우 효과 추가.

## 5. 성공 지표
- `package_scene.png` 이미지와 시각적 레이아웃이 90% 이상 일치함.
- 카테고리별로 패키지가 명확히 구분되어 표시됨.
- 각 패키지의 진행률이 정확히 표시됨 (현재는 0으로 고정 가능).
