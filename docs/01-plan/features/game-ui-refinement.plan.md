# 계획서: 게임 화면 UI 고도화 (Flow Free 스타일)

## 1. 개요
제공된 이미지(`game_scene.png`)를 분석하여 현재의 게임 플레이 화면을 상용 수준의 세련된 UI/UX로 고도화합니다. 헤더, 통계 바, 그리드 스타일 및 하단 컨트롤러를 전면 개편합니다.

## 2. 핵심 디자인 요소 (이미지 분석 반영)
- [ ] **헤더 (Top Header)**:
    - 좌측: 원형 뒤로가기 버튼 (`createCircularBackButton` 활용).
    - 중앙: "레벨 1 5x5" 형식의 정보 표시 (녹색 텍스트).
    - 우측: 톱니바퀴 설정 아이콘.
- [ ] **통계 바 (Stats Bar)**:
    - 그리드 상단에 3개 영역으로 구분된 텍스트 배치.
    - 좌측: "흐름: N/M" (연결된 색상 수).
    - 중앙: "무브: N 최고: M" (현재 이동 횟수 및 최고 기록).
    - 우측: "파이프: N%" (보드 점유율).
- [ ] **그리드 고도화 (Grid Styling)**:
    - **셀 배경**: 경로가 지나는 셀은 해당 색상의 어두운 버전으로 배경을 채움 (Glow 효과).
    - **가이드 라인**: 셀 경계에 아주 은은한 외곽선 추가.
- [ ] **하단 컨트롤 바 (Bottom Controls)**:
    - 좌측: 실행 취소(Undo) 버튼 (원형 화살표).
    - 중앙: 힌트 시스템 (전구 아이콘, 남은 힌트 수 표시).
    - 우측: 레벨 초기화(Reset) 버튼 (회전 화살표).

## 3. 기술 스택 및 구현 전략
- **UIManager 확장**: 헤더, 통계 바, 하단 버튼 전용 메서드 추가.
- **GridManager 수정**: 셀 배경색 채우기 로직 (`drawCellBackground`) 추가.
- **MainScene 수정**: 새로운 UI 레이아웃 적용 및 하단 버튼 이벤트 바인딩.

## 4. 상세 작업 목록
- [ ] **UIManager.js 확장**:
    - `createGameHeader()`: 레벨 정보 및 설정 아이콘 배치.
    - `createStatsBar()`: 흐름, 무브, 파이프 점유율 텍스트 배치.
    - `createGameControls()`: 하단 Undo, Hint, Reset 버튼 배치.
- [ ] **GridManager.js 고도화**:
    - `draw()` 메서드에서 셀 배경색(경로 색상의 0.2 alpha)을 칠하는 로직 추가.
- [ ] **MainScene.js 리팩토링**:
    - 9:16 비율에 맞춘 각 요소의 수직 위치 조정.
    - 실시간 통계 업데이트(`updateHUD`) 로직 보완.
- [ ] **인터랙션 추가**: 하단 버튼들에 대한 클릭 피드백 및 기능 연결.

## 5. 성공 지표
- `game_scene.png` 이미지와 시각적 레이아웃이 95% 이상 일치함.
- 경로가 지나는 셀의 배경색이 실시간으로 자연스럽게 업데이트됨.
- 모든 버튼이 모바일 환경에서 터치하기 편한 크기로 배치됨.
