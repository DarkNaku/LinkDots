# 계획서: LinkDots (Flow Free 클론)

## 1. 개요
동일한 색상의 점들을 그리드 위에서 연결하는 미니멀한 퍼즐 게임입니다. 모든 점의 쌍을 연결해야 하며, 완벽한 승리를 위해서는 그리드의 모든 칸이 선(파이프)으로 채워져야 합니다.

## 2. 핵심 기능
- [ ] **그리드 및 보드 시스템**: 
    - 가변 그리드 크기 (초기 5x5).
    - 좌표 기반 셀 관리.
- [ ] **점(Dot) 배치**: 
    - 고정된 위치에 동일한 색상의 점 쌍 배치 (레벨 데이터).
- [ ] **선 그리기 (상호작용)**: 
    - 터치 또는 마우스 드래그를 통해 점 사이의 연속적인 선 생성.
    - 선의 색상은 시작점의 색상과 일치해야 함.
- [ ] **로직 및 규칙**:
    - [ ] **교차 금지**: 선은 서로 겹칠 수 없음. 기존 선을 가로지르면 기존 선이 끊어짐.
    - [ ] **동일 색상 연결**: 동일한 색상의 점끼리만 연결 가능.
    - [ ] **전체 점유**: 승리를 위해 그리드의 모든 칸이 선으로 채워져야 함.
- [ ] **UI/UX 요소**:
    - [ ] **게임 보드**: 어두운 배경에 고대비 색상의 점.
    - [ ] **상태 표시(HUD)**: 레벨 이름, 이동 횟수(Moves), 보드 점유율(%).
    - [ ] **컨트롤**: 레벨 초기화(Reset) 버튼.
    - [ ] **피드백**: 연결 시 효과음 및 승리 애니메이션.

## 3. 기술 스택
- **프레임워크**: Phaser 3 (루트의 phaser.js 활용).
- **언어**: JavaScript (ES6+).
- **에셋**: 절차적 그래픽 (원형 점, 사각형/직선 파이프).

## 4. 상세 작업 목록
- [ ] 5x5 그리드의 Phaser 게임 인스턴스 초기화.
- [ ] 데이터 관리를 위한 `Grid` 및 `Cell` 클래스 생성.
- [ ] 경로 그리기 로직 구현 (입력 처리, 라인 렌더링).
- [ ] 경로 교차 시 기존 선 끊기(Pipe Breaking) 로직 구현.
- [ ] 승리 조건 판정 로직 (모든 점 연결 + 모든 칸 점유).
- [ ] 초기화 버튼 및 이동 횟수 카운터 추가.
- [ ] 3~5개의 샘플 레벨 데이터 구축.

## 5. 성공 지표
- 사용자가 버그 없이 모든 점의 쌍을 연결할 수 있음.
- 완벽한 승리 시 그리드가 100% 채워짐을 감지함.
- 경로 그리기가 매끄럽고 그리드 좌표를 정확히 따름.
- 경로 중복 시 기존 경로가 정상적으로 제거됨.
