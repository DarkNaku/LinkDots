# 계획서: 레벨 선택 화면 UI 고도화 (Flow Free 스타일)

## 1. 개요
제공된 이미지(`level_scene.png`)의 디자인을 분석하여, 현재의 기본적인 레벨 선택 화면을 상용 게임 수준의 세련된 UI로 고도화합니다.

## 2. 핵심 디자인 요소 (이미지 분석 반영)
- [ ] **헤더 (Header)**:
    - 좌측 상단: 원형 배경의 뒤로가기 화살표 버튼.
    - 중앙 상단: 현재 선택된 패키지 이름 (예: "클래식 팩") 표시.
- [ ] **서브 타이틀**:
    - 그리드 바로 위에 현재 팩의 상세 정보 표시 (예: "5x5 - 쉬움").
- [ ] **레벨 그리드 (Level Grid)**:
    - **정사각형 타일**: 둥근 모서리가 없는 날카로운 정사각형 또는 아주 미세한 라운드.
    - **테두리 스타일**: 
        - 미완료: 해당 팩의 테마 색상(예: 녹색) 테두리 + 중앙에 흰색 번호.
        - 완료: 테마 색상으로 꽉 찬 배경 (이미지에는 없으나 일반적인 Flow Free 규칙 반영).
    - **배치**: 5열 그리드 시스템.
- [ ] **페이지네이션 (Pagination)**:
    - 하단 중앙에 현재 페이지 위치를 나타내는 작은 점(Dot)들 표시.
- [ ] **배경 (Background)**:
    - 완전한 검은색 배경에 은은한 색상 그라데이션(Glow) 효과 추가.

## 3. 기술 스택 및 구현 전략
- **UI 구성**: Phaser 3 `Graphics` 및 `Text` 객체 활용.
- **레이아웃**: `LevelSelectScene.js`의 그리드 계산 로직 정밀 조정.
- **인터랙션**: 타일 클릭 시 미세한 밝기 변화 및 씬 전환 페이드 효과 유지.

## 4. 상세 작업 목록
- [ ] **UIManager 확장**: 
    - `createCircularBackButton()`: 원형 테두리가 있는 뒤로가기 버튼 메서드 추가.
    - `createStyledGridButton()`: 이미지와 동일한 테두리 스타일의 레벨 버튼 구현.
- [ ] **LevelSelectScene 리팩토링**:
    - 헤더와 서브 타이틀 레이아웃 조정.
    - 5x6 (또는 데이터에 따른) 그리드 배치 정교화.
- [ ] **페이지네이션 UI 추가**: 하단에 레벨 페이지를 나타내는 인디케이터 구현.
- [ ] **배경 고도화**: 하단 광고 영역을 제외한 나머지 영역에 네온 느낌의 배경 처리.

## 5. 성공 지표
- `level_scene.png` 이미지와 시각적으로 90% 이상 일치함.
- 레벨 번호가 가독성 있게 표시되며, 터치 영역이 명확함.
- 패키지 이름과 상세 정보가 상단에 정확히 표시됨.
